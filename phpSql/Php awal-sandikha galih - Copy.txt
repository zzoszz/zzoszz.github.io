Kenapa PHP untuk pemrograman web?
1. Php relatif mudah untuk pemula
2. cuma butuh tahu html
3. gratis dan open source
4. banyak dukungan komunitas (coba tulis saja php di google)
5. dokumentasi lengkap, jadi kalau ada masalah banyak penyelesaiannya
6. dibuat khusus untuk web
7. most popular server side programming languages, kebanyakan digunakan website (php, asp.het, java, static files, couldfusion)
8. banyak website besar yang dibuat php, setidaknya awalnya dengan php (facebook, yahoo, wikipedia, fleckr)
9. content management system yang menggunakan php sangat banyak contoh (wordpress, joomla, drupal, moodle)
10. bnyak frameword contoh: cakephp, codeigniter, yiiframework, laravel, symfony)

8. database
9. CRUD (create, read, update, delete)
10. login dan registrasi
11. session dan cookie
