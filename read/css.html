Unit Value

  • CSS value dan unit (nilai dan satuan)
  • value: integer (+/-), number(angka/pecahan), percentage(%), dimension (number that has a velue)
  • dimension: length, angle, time, resolution angle: deg degree (360deg), rad radian (2pi), grad gradian (400g), turn(1)
  • time: s second, ms mili second.
  • resolution: dpi dot per inch, dpcm dot per centimeter, dppx dot per pixel.
  • length: absolute (px pixel, pt point, pc pica, cm, mm, in) and relative.
  • relative: berikut:
  • % persentase ukuran relatif terhadap ukuran elemen parent
  • em relatif terhadap ukuran font yang sedang digunakan
  • rem relatif terhadap ukuran font root html / :root
  • ch relatif terhadap lebar karakter '0' dari font yang sedang digunakan
  • vh relatif terhadap tinggi dari viewport
  • vw relatif tarhadap lebar dari viewport
  • vmin relatif terhadap dimensi terkecil sebuah viewport
  • vmax relatif terhadap dimensi terbesar dari sebuah viewport
