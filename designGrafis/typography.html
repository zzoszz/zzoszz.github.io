Typography

Pengantar

Tipography adalah seni memperindah tampilan huruf, meliputi font style, layout dan struktur, tujuannya memperoleh emosi tertentu dan menyampaikan pesan tertentu. Tipografi baik membentuk hirarki visual kuat, memberi keseimbangan grafis ke dalam desain interface, optimasi user membaca dan akses sehingga user experience-nya baik. Berikut kategori font-family:

  • Serif (memiliki goresan kecil di tiap ujungnya, sifatnya klasik dan tradisional, umum di media cetak dan surat kabar: berikut jenisnya:
    1. Times New Roman
    2. Georgia
    3. Palatino
    4. EB Garamond
  • Sans Serif (tidak memiliki goresan kecil di ujungnya, lebih bersih dan modern dibanding serif, umumnya di media digital.
    1. Arial
    2. Helvatica
    3. Calibri
    4. futura
  • Monospace fonts - here all the letters have the same fixed width. They create a mechanical look.
    1. Courier New
    2. Lucida Console
    3. Monaco
  • Cursive fonts imitate human handwriting.
    1. Brush script mi
    2. Lucida handwriting
  • Fantasy fonts are decorative/playful fonts.
    1. Copperlate
    2. Papyrus
  • Font Sleb Serif (bentuk hampir sama dengan serif tapi lebih tebal, berani dan tegas. Sering digunakan oleh perusahaan elektronik dan produsen mobil untuk mengkomunikasikan rasa kekuatan dan maskulin dalam branding mereka)
    1. ROckwell
    2. Detroit
    3. Derius
    4. BW Glenn Slab
  • Font Script (tegak bersambung dan elegan, umumnya digunakan pada tulisan singkat bukan paragraf panjang, tidak bisa dengan huruf kapital semua agar menjaga tulisan tetap tersambung)
    1. Bromello
    2. Brush Script mi
    3. Palace script mi

Tips

  • Tonjolkan teks menggunakan hierarchy: untuk memaksimalkan keterbacaan teks, prakteknya buat ukuran heading 1.618 kali belih besar dari body teks
  • Kombinasi jenis font atau typeface: Misal: judul dengan calibri, lainnya roboto. Jangan banyak font: judul Noto serif, paragraf roboto, cancel Signika, Ok varta
  • Gunakan kontras warna yang berimbang, tipsnya:
    1. Teks kecil harus memiliki kontras minimal 4.5:1 terhadap backgroundnya.
    2. Teks besar (ukuran 18px ke atas) harus memiliki kontras minimal 3:1 terhadap backgroundnya.
    3. Kunjungi webm atau gunakan plugin contrast
  • Hindari penulisan menggunakan huruf besar semua
  • White space (negative space) adalah area kosong antar elemen di dalam sebuah komposisi design. Para pembaca biasanya tidak menyadari pentingnya whitespace

Font Pairing

FOnt Pairing rules
  1. Complement: it is always safe to find font pairings that complement one another. A great font combination should harmonize without being too similar or too different
  2. Use font superfamilies: A font superfamily is a set of fonts design to work well together. So, using different fonts within the same superfamily is safe. For example, the Lucida superfamili contains the following fonts: lucida sans, lucida serif, lucida typewriter sans, lucida typewriter serif and lucida math.
  3. Contrast is king: two fonts that are too similar will often conflict. However, contrast, done the right way brings out the best in each font. Example combining serif with sans serif is a well known combination. A strong superfamily includes both serif and sans serif variations of the same font (e.g. Lucida and lucida sans)
  4. Choose only one boss: One font should be the boss. This establishes a hierarchy for the fonts on you page. This can be achieved by varying the size, weight and color.
