Alasan menggunakan codeIgniter:

  1. Ukuran CI sangat kecil sekali
  2. Dokumentasi yang jelas dan lengkap
  3. Konfigurasi simpel
  4. Performa yang sangat cepat
  5. Berbasis open source
  6. Prerequest: HTMP dasar, PHP dasar, OOP

Tools utamanya: xampp, folder codeinginter, text editor, web browser berikut tahapan memulai codeIgniter:

  1. konfigurasi: masuk folder application > config > (ada tiga file yang digunakan: autoload, config, database, routers
  2. autoload: perhatikan libraries: array('database', 'session', 'form_validation' dan ada banyak lagi libraries yang digunakna codeIgniter); helper: array('url', 'form'); Model
  3. config['base_url'] = 'localhost/codeigniter3113/';
  4. database
  5. routers: default controllernya 'wecome', kita bisa merubah default controllernya di folder controllers > file welcome. Misalnya kita copy file welcome, dan paste dengan nama 'contoh.php' (ubah isi public function index() dengan echo'h1 ini adalah contoh /h1') simpan. Lalu kembali 'routers', ganti 'welcome' dengan 'contoh'
  6. Perbedaan cara membuka localhost biasa dan localhost codeigniter
    • biasa: localhost/biasa/index.php
    • codeigniter: localhost/codeigniter/index.php/nama_class/index
    • penjelasan:
      • localhost: alamat server
      • codeigniter: nama direktori
      • index.php: nama controller default CI
      • welcome: nama controller/class (URI segmen 1)
      • index: nama method/ fungsi dalam controller (URI segmen 2)
  7. Berikut perbedaan URI dan URL
    • URL: Uniform Resource Locator, identifikasi sumber web menggunakan lokasi
    • URI: Uniform Resource Identifier, identifikasi sumber web menggunakan nama dan lokasi
