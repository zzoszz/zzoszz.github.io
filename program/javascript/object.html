Object

Pengantar

Objek bisa menyimpan variabel, array, objek lain dan fungsi. Yang menyimpan nilai variabel atau array disebut property sedangkan yang menyimpan fungsi disebut method berikut contohnya

this dalam function atau nested function hasilnya akan sama (lingkupnya window), this dalam object hasilnya akan (lingkupnya object)

            pertama lakukan:
            console.log(this);
            console.log(window);
            let objek = {
                console.log(this);
                var: 'variabel',
                arr: ['arr1', 'arr2', 'dst'],
                objChild: {
                    child1: 'satu',
                    child2: 'dua',
                }
                met: function(){
                    return this.variabel;
                }
            }
            console.log(objek);
            Berikut cara lain membuat objek:
            let objek = {}
                objek['var'] = 'variabel'; atau
                objek.var = 'variabel'
                objek['arr'] = ['arri', 'arr2', 'dst']; atau
                objek.arr = ['arri', 'arr2', 'dst'];

Berikut cara memanggilnya:

this dalam function atau nested function hasilnya akan sama (lingkupnya window / mengembalikan objek global), this dalam object hasilnya akan (lingkupnya object / mengembalikan objek lokal)

Objek literal

Function Declaration

PROBLEM: kalau melakukan instansiasi pada objeknya, ternyata di belakang layar, method tetap dibuat (mau dipakai atau tidak) tetap disimpan ke dalam memori, solisi agar lebih efektif adalah dengan membuat objek terpisah

PROBLEM LAIN: kalau membuat method baru, maka object declarasiannya juga harus disesuaikan, maka yang dikelola ada dua
