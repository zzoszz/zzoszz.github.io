Pengantar

sass w3schools | sass | alasan dibuatnya sass karena dirasa ada kekuarngan pada css berikut: tidak ada variabel, function, modularisasi, tidak bisa melakukan operasi dan logika pemrograman dan banyak duplikasi kode. Solusi dari pre processor: variable, mixin, import dan uses, operator dan built in function, nested css. Alasan menggunakan sass: mempermudah penulisan, pengolahan, memiliki banyak fungsi built-in, kita dapat membuat fungsi sendiri, dapat memodifikasi framework css
alasan tidak menggunakan sass: Banyak fitur sass yang sekarang sudah dimiliki css seperti variabel, operator; tidak untuk projek kecil; styled components / css in js

ketika mengedit style.scss lalu 'watch sass' (di bawah), maka akan ada dua file baru yaitu style.css dan style.map.css. Jika kamu ingin agar dua file itu di folder lain maka lakukan hal berikut:

  1. file/preferences/setting(ctrl+,)
  2. cari 'live sass compiler'
  3. scroll hingga ke 'formats' lalu klik edit in settings.json
  4. savePath: null, nullnya diganti nama folder misal 'css', nanti filenya akan tersimpan di folder itu
